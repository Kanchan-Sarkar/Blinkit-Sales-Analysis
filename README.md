# Blinkit-Sales-Analysis
1. Project Title: Blinkit Sales Performance, Customer Satisfaction and Inventory Distribution Analysis.
2. Short Describtion:
   The Blinkit Sales Analysis Dashboard delivers key insights into sales performance across products, outlet types, and locations. It highlights trends in total sales, average sales, and customer ratings, helping identify top-performing stores and high-demand categories. This dashboard supports data-driven decision-making for sales optimization, market expansion, and improving overall business efficiency.
  A. Purpose of the Dashboard:
   The main purpose of this dashboard is to:
      1.Monitor overall sales performance across different outlet sizes, types, and locations.
      2.Understand product demand based on item types and fat content.
      3.Compare outlets by establishment year, size, and location tier.
      4.Identify trends in sales and performance over time to support data-driven decision-making for marketing, inventory, and expansion.
  B. Key Metrics Displayed:
     1.Total Sales: ₹1.20M — overall revenue generated.
     2.Average Sales: ₹141 — average revenue per item/outlet.
     3.Number of Items: 8,523 — total items analyzed.
     4.Average Rating: 3.9 — customer satisfaction indicator.

3. Dashboard Components:
    1.KPIs (Top Section)
      Quick summary cards for Total Sales, Average Sales, Number of Items, and Average Rating.
      These KPIs provide an at-a-glance view of performance.
    2. Outlet Establishment (Line Chart)
      Shows sales trend from 2010 to 2020.
      Peak sales (~₹205K) occurred around 2018, followed by a dip.
      Highlights how outlet age influences sales.
    3. Fat Content Analysis (Donut Chart)
      Compares Low Fat vs Regular items.
      Regular items contribute more to total sales (~₹776K) compared to low-fat (~₹425K).
    4. Item Type Breakdown (Bar Chart)
      Displays sales by product category (e.g., Fruits, Snack Foods, Dairy, Household, etc.).
      Top categories: Fruits and Vegetables, Snack Foods, and Household items.
    5. Fat by Outlet (Bar Chart)
      Compares fat content sales across Tier 1, Tier 2, and Tier 3 outlets.
      Tier 3 outlets show the highest sales (~₹0.31M).
    6. Outlet Size (Pie Chart)
      Medium-sized outlets dominate sales (~₹507.9K), followed by small and high outlets.
    7. Outlet Location (Bar Chart)
      Tier 3 locations lead with the highest total sales (~₹472K), followed by Tier 2 and Tier 1.
    8. Outlet Type Summary (Table)
      Tabular view showing Total Sales, Number of Items, Average Sales, Rating, and Visibility per outlet type.
      Supermarket Type 1 and Type 3 are top performers in sales volume.
    9. Filters (Left Panel)
       Slicers allow filtering data by:
          Outlet Size
          Item Type
          Outlet Location Type
This interactivity enables customized insights for specific business questions.

4. Tech Stack:
   This dasbihboard was build using the following tools & technologies:
     *PowerBI Desktop - Main data visualization platform used for repord creation.
     *Power Query - Data transformation and cleaning layer for reshaping the data.
     *DAX(Data Analysis Expressions): Used for calculated measures, dynamic visuals and conditional logic.
     *Data Modeling - Relationships established among tables to enable cross-filtering and aggregation.
     *File Format - .pbix for development and .png for dashboard previews.

5. Conclusion:
     The Blinkit Sales Analysis Dashboard effectively summarizes sales and outlet performance across multiple dimensions.
It empowers managers and analysts to:
  -Identify high-performing outlet types and regions,
  -Optimize inventory based on product category demand, and
  -Plan targeted marketing or expansion strategies based on sales and customer ratings.
